# ❄️ NixOS

🇵🇱
Jeżeli chcesz wgrać config do siebie to:  
Jako admin, skopiuj wszystkie pliki za wyjątkiem hardware-configuration.nix do folderu /etc/nixos  
Otwórz edytorem tekstu configuration.nix i podmień słowo 'jakub' twoim kontem użytkownika  
Otwórz edytorem tekstu services.nix i podmień słowo 'jakub' twoim kontem użytkownika  
  
Później wywołaj w terminalu  
sudo nixos-rebuild boot
